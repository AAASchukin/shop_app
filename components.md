# Список Компонентов

## 1. Сервис Пользователей
### API:
- Создание Нового Пользователя (регистрация)
    - Входные параметры: идентификатор, логин, имя, фамилия
    - Выходные параметры: отсутствуют
- Поиск Пользователя по Логину
    - Входные параметры: логин
    - Выходные параметры: имя, фамилия
- Поиск Пользователя по Маске Имени и Фамилии
    - Входные параметры: маска фамилии, маска имени
    - Выходные параметры: логин, имя, фамилия

## 2. Сервис Товар
### API:
- Создание Товара
    - Входные параметры: идентификатор, название товара, артикул, количество
    - Выходные параметры: идентификатор товара
- Получение Каталога всех Товаров
    - Входные параметры: отсутствуют
    - Выходные параметры: массив товаров, в каждой указаны его идентификатор, название, артикул, количество

## 3. Сервис Корзина
### API:
- Создание Корзины
    - Входные параметры: идентификатор пользователя, идентификатор товара
    - Выходные параметры: идентификатор товара, название, артикул, количество
- Получение Товара для Пользователя
    - Входные параметры: пользователь (логин)
    - Выходные параметры: массив товаров (идентификатор, название, описание, стоимость)
- Добавление Товара в Корзину
    - Входные параметры: идентификатор товара (tovar_id), (идентификатор, название, описание, стоимость)
    - Выходные параметры: обновлённый заказ в корзине: идентификатор товара (tovar_id), список товаров, дата создания (date)

# Назначение системы
| Система | Описание |
| ------- | -------- |
| Сайт магазина с добавлением товаров в корзину | Веб-интерфейс, назначение- обеспечивает доступ к информации магазина по товарам и добавление их в корзину. Бэкенд и фронтэнд сервиса реализован в виде сервисной архитектуры |

## Наименование: Сайт магазина с добавлением товаров в корзину
### Описание:
Приложение должно содержать следующие данные:
- Пользователь
- Товар
- Корзина

### Реализовать API:
- Создание нового пользователя
- Поиск пользователя по логину
- Поиск пользователя по маске имя и фамилии
- Создание товара
- Получение списка товаров
- Добавление товара в корзину
- Получение корзины для пользователя
